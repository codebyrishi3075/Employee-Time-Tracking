# Employee Project Cost Tracker

### Features
- Track **employee hours** per project.
- Automatic **budget monitoring** with warnings.
- Suggest employee **reassignment** when budgets exceed limits.
- **Sorted project view** based on total hours.
- **Recursive check** to identify budget-violating employee.
- **Shallow vs deep copy** demonstration.
- Supports **interactive mode** and **demo mode**.
- Uses **generators**, **nested dictionary comprehensions**, and **set comprehensions**.
- Includes utility logging using timestamps.

### How to Run
```bash
python self_project.py
```

## Modes
### 1. Interactive Mode
- Add employees, projects, and hours.
- View summaries and suggestions.

### 2. Demo Mode
- Runs with pre-filled sample data.

## File Source
Project code located in: `self_project.py`

- Default project budget is **150 hours**, customizable during entry.
